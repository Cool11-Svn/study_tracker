This program is called study tracker - it allows you to create a list with some data about particular subject.

## ⏱ Features

- Add new study sessions
- View all saved sessions
- Edit existing sessions
- Delete sessions
- Calculate total study time in hours
- Calculate weekly and daily study time based on a selected date

The program creates a json file so do not try to manually edit it - or you will get crash.
